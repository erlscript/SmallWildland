### 小荒原 

"小荒原"是一个多人在线 H5 游戏。

> 其中很多游戏素材来自于这位 [Sithjester 老哥的免费游戏素材网站](http://untamed.wild-refuge.net/rpgxp.php)。
> w a s d q e z x 移动。
> 空格键说话，p 键捡东西。
> j k l 使用，切换， 丢下武器。
> u i o 使用，切换， 丢下药剂。
